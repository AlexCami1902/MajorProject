## Reporting a Vulnerability

If you find a security vulnerability in the Cricket Scoring App, please report it as soon as you can. To do so:

1. **Contact:** Send an email to alexandercamilleri@student.riverview.nsw.edu.au with a detailed description of the issue.
2. **Response Time:** You can expect an initial response within 48 hours.
3. **Evaluation:** If the vulnerability is accepted, a patch will be developed, tested, and released promptly.
4. **Status Updates:** You will receive updates on the progress of the fix at least once a week until resolution.
5. **Declined Reports:** If the issue is deemed not to be a vulnerability, an explanation will be provided.

We appreciate disclosure and will acknowledge contributions that help improve the security of the project.
